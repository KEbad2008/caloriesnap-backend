# ⚡ CalorieSnap
 
CalorieSnap lets you upload a photo of any meal and instantly get an AI-powered estimate of its calories, protein, carbs, and fat. Simply create an account, snap or upload a food photo, and hit Analyze. Each user gets 3 free scans per day which resets at midnight. Built with React, Node.js, Supabase, and the Anthropic Claude AI model.
 
🔗 Live at: https://caloriesnapv1.netlify.app
